콘티 → 이미지 선택 → 슬라이드 생성 → 가사 2줄 자동 배분

1

콘티 입력

곡명 뒤에 슬라이드 수를 적어주세요. 예: 내 모습 이대로 (4)

2

이미지 슬라이드 선택

이미지를 여러 장 선택하면 파일 이름 순서대로 슬라이드에 들어갑니다. 01, 02, 03처럼 이름을 맞추면 좋습니다.

선택된 이미지 없음
3

슬라이드 만들기

콘티의 슬라이드 수에 맞춰 슬라이드를 만들고, 선택한 이미지를 순서대로 넣습니다.

가사 가져오기 / 2줄씩 자동 입력

네이버에서 가사를 찾아 복사한 뒤 아래에 붙여넣으면, 선택한 곡의 슬라이드에 2줄씩 자동으로 들어갑니다.

슬라이드별 이미지와 가사 편집

각 이미지에 맞춰 가사를 확인하고 수정하세요. 후렴은 체크하면 첫 가사 인식 시 이동 대상으로 사용됩니다.
